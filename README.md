# Untold Words

An anonymous message-sharing website where users can send untold, unsaid, and unsent words.

## About
Untold Words is a simple frontend web project that allows people to express their thoughts anonymously, without login or identity.

## Built With
- HTML
- CSS
- JavaScript
- Firebase

## Features
- Anonymous message submission
- Clean and minimal UI
- Frontend-only project

## Purpose
This project was built to practice frontend development and create a safe space for anonymous expression. This was made to showcase my skills as well as for others to showcase their untold words.
